# Music Bridge 0.9.0

Первая предварительная Windows-сборка перед выпуском 1.0.

## Скачать

- **Music.Bridge.Setup.0.9.0.exe** — рекомендуемый вариант: установщик Windows x64.
- **Music.Bridge.Windows.x64.zip** — portable-версия без установки.

Python и ручная установка зависимостей не требуются. Для работы приложения необходим Google Chrome.

Сборка пока не подписана коммерческим сертификатом, поэтому Windows SmartScreen может показать предупреждение. Для запуска выберите «Подробнее» → «Выполнить в любом случае».

## Что уже работает

- импорт публичного плейлиста по одной ссылке без Client ID и авторизации;
- получение названия, владельца, продолжительности и списка композиций;
- автоматическое создание папки с безопасным названием плейлиста;
- ручная смена папки назначения;
- проверка списка и возврат к другому плейлисту;
- сопоставление с отдельно подключённым лицензированным каталогом;
- параллельная обработка, пауза, продолжение и отмена;
- прогресс и отдельный статус каждой композиции;
- итоговый отчёт и открытие папки с результатом;
- локальные настройки, журналы и ручной диагностический архив;
- тёмный интерфейс с масштабированием под разные размеры экрана.

Музыка не загружается со стримингового сервиса: публичная ссылка используется только для чтения метаданных. Каталог должен использоваться при наличии необходимых прав.
