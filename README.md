# Music Bridge

Music Bridge — настольное приложение для Windows, которое переносит публичный плейлист в локальную музыкальную коллекцию.

Приложение получает из ссылки название плейлиста и список треков, сопоставляет их с отдельно подключённым лицензированным каталогом и сохраняет доступные композиции в выбранную папку. Аудио не загружается со стримингового сервиса: ссылка используется только для чтения общедоступных метаданных.

## Скачать

Откройте раздел [Releases](../../releases/latest) и скачайте **Music Bridge Setup 0.9.0.exe**.

Это обычный установщик для Windows 10/11 x64. Python и ручная установка зависимостей не нужны. Также в релизе есть portable-архив **Music Bridge Windows x64.zip**, который можно распаковать и запустить без установки.

> Сборка пока не подписана коммерческим сертификатом. Windows SmartScreen может показать предупреждение: выберите «Подробнее» → «Выполнить в любом случае». Контрольные суммы файлов опубликованы в описании релиза.

## Как пользоваться

1. Установите и запустите Music Bridge.
2. Вставьте публичную ссылку на поддерживаемый плейлист.
3. Проверьте найденные композиции и папку назначения.
4. Запустите обработку и дождитесь итогового отчёта.

Для получения метаданных публичного плейлиста не нужны Client ID, пароль или вход в аккаунт. Для обработки требуется Google Chrome. Музыкальный каталог должен быть настроен владельцем сборки и использоваться только при наличии необходимых прав.

## Данные и приватность

- телеметрия и скрытая отправка данных отсутствуют;
- настройки хранятся локально в `%APPDATA%\Music Bridge`;
- журнал работы и временные файлы хранятся в `%LOCALAPPDATA%\Music Bridge`;
- диагностический архив создаётся только вручную пользователем.
